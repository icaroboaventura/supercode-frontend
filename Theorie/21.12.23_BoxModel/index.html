BOX Model

beschreibt wie sich Block Elemente verhalten

Der Platzbedarf errechnet sich aus:

  1. width/height (Content-Box)
  2. padding (Padding-Box)
  3. border (Border-Box)
  4. margin (Margin-Box)

Box-sizing property

hilft uns Box Model zu manipulieren

box-sizing:content-box => default Wert, Boxen werden auf addiert

box-sizing:border-box => Breite wird automatiasch mit Border verrechnet

Display property:inline, block, inline-block

inline: Elemente können Höhe und Breite annehmen

block: nehmen 100% der Breite ein

inline-block: kombiniert das Beste von beidem, man hat nicht immer 100% Breite & man kann trotzdem Höhe/Breite vergeben
